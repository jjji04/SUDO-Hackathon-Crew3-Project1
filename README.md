# 📅 스마트 일정 관리 비서 (Smart Planner AI)
> **AI 기반의 효율적인 일정 관리 및 할 일 목록 자동화 솔루션**  
> 본 프로젝트는 **GitHub을 활용한 팀 협업 방식과 Git 워크플로우(Gitflow)** 숙달을 핵심 목표로 합니다.

---

- **주요 목표**:

- **Collaboration**: GitHub, Git, KakaoTalk

## 🔄 협업 워크플로우 (Git Flow)
본 팀은 원활한 협업을 위해 다음과 같은 Git 전략을 사용합니다.

1. **Issue 생성**: 작업 시작 전 작업 내용을 Issue에 등록합니다.
2. **Branch 전략**: `main` <- `feature/기능명` 순으로 브랜치를 관리합니다.
3. **Pull Request (PR)**: 기능 구현 완료 후 팀원들의 코드 리뷰를 거쳐 `main`에 머지합니다.

## ✨ 주요 기능
- **일정 표 정리**: 텍스트 입력을 통해 복잡한 일정을 Markdown 표 형식으로 자동 변환합니다.
- **할 일 목록 우선순위**: 중요도에 따라 이모지(🔴🟡🟢)로 할 일을 구분합니다.
- **실시간 타이핑 효과**: AI 비서가 답변을 작성하는 듯한 생동감 있는 인터페이스를 제공합니다.
- **대화 기록 유지**: 세션 상태 관리를 통해 대화의 맥락을 기억하며, 필요 시 초기화가 가능합니다.
